// gMap компонент

initMap(items, options) - получаем данные для инициализации компонента, настройки, проверка на повторную инициализацию

renderMap(locations, showRegion[= true]) - получает массив объектов с координатами и описанием локации, при поднятом флаге показывает маркеры с регионами доступа

getLocation() - получает местонахождение пользователя, вызов коллбэка при успехе/неудаче

_geolocationSuccess(position) - внутренний метод при удачном определении пользователя - отрисовка маркера пользователя на карте

_geolocationFailure - внутренний метод при неудачном определении позиции пользователя

showDestination(locations, userLocation, showAll = false) - отображает на карте зоны доступности для точки, принимает локации, положение пользователя и отображает 
зону доступа, в которой он находится. Если поднять флаг - покажет все зоны для всех локаций

_isInDestination(center, radius, point) - определяем находится ли точка в радиусе обслуживания локации, принимает локацию, радиус обслуживания, текущую точку, return true||false

 _setUser(location) - отображаем локацию пользователя на карте в точке с переданными координатами

_getCenter(locations) - внутренний метод - определение центра карты - географический центр массива локаций

_isPoint(location) - внутренний метод - проверка объекта на наличие координатат

 Пример
 Передаем в компонент набор локаций:

 let locations =
        [
                {
                    lat: 43.5868685,
                    lng: 39.7299474,
                    description: 'первая точка',
                    img: 'some url',
                    radius: 500,
                },
                {
                    lat: 43.6085269,
                    lng: 39.7313797,
                    description: 'вторая точка',
                    img: 'some url',
                    radius: 500,
                },
                {
                    lat: 43.6191221,
                    lng: 39.7218364,
                    description: 'третья точка',
                    img: 'some url',
                    radius:500,
                },
                {
                    lat: 43.593971,
                    lng: 39.736433,
                    description: 'четвертая точка',
                    img: 'some url',
                    radius: 500,
                },
        ];

вызываем метод:

		let m = gMap;
        m.init();
        m.renderMap(locations);
        m.showDestination(locations, null, true);

В тэге с id='#map' отрисовываеся карта с переданными локациями и положением пользователя.
