// gMap компонент

initMap(items, options) - получаем данные для инициализации компонента, настройки, проверка на повторную инициализацию

renderMap(locations) - получает массив объектов с координатами и описанием локации

_isPoint(location) - внутренний метод - проверка объекта на наличие координатат

_geolocationSuccess(position) - внутренний метод при удачном определении пользователя - отрисовка маркера пользователя на карте

_geolocationFailure - внутренний метод при неудачном определении пользователя

_getCenter(locations) - внутренний метод - определение центра карты - географический центр массива локаций

_isInDestination(center, radius, point) - внутренний метод - определяем находится ли точка в радиусе обслуживания локации, принимает локацию, радиус обслуживания, текущую точку, return true||false

 _setUser(location) - внутренний метод - отображаем локацию пользователя на карте в точке с переданными координатами
